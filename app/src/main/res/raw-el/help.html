Πώς να χρησιμοποιήσετε?
  1. Σύνδεση με το προεπιλεγμένο λογαριασμό: manager / 123456
  2. Πριν από την έναρξη εκτέλεσης παραγγελιών, θα πρέπει να εγκαταστήσετε τα τραπέζια, τις κατηγορίες, τα προϊόντα και σχόλια για το εστιατόριό σας στις ρύθμισεις.
  3. Για εκτύπωση αποδείξεων, θα πρέπει να ρυθμίσετε τον εκτυπωτή στο μενού ρυθμίσεων. Σιγουρευτείτε τη σωστή ρύθμιση του εκτυπωτή.
  4. Μετά τη ρύθμιση, μπορείτε να ανοίξετε το τραπέζι, πέρνεται παραγγελίες, και να στέλνετε παραγγελίες στον κουζίνα / μπαρ εκτύπωση.
  5. Αφού στείλετε παραγγελίες, μπορείτε να ελέγχετε τις παραγγελίες κάνωντας κλίκ στο τραπέζι, εξοφλήτε παραγγελίες και εκτυπώνετε αποδείξεις.
Τραπέζια
  1. Τραπέζια με χρώμα
    1. Κάντε κλικ στο τραπέζι για την οθόνη του μενού.
  2. Δωρεάν πίνακα
    1. Κάντε κλικ στο τραπέζι για νέα παραγγελεία.
  3. Κάντε κλικ στο κουμπί του μενού της συσκευής, για reports, ρύθμισεις, βάση δεδομένων.
Παραγγελία
  1. Κάντε κλικ στο όνομα του Προϊόντος παραγγελία για να αλλάξει τιμή, προσθέστε παρατήρηση και την εκμετάλλευση.
  2. Κάντε κλικ στο "Επιβεβαίωση" για να επιβεβαιώσει τα προϊόντα.
  3. Κάντε κλικ στο κουμπί "+" για να προσθέσετε προϊόντα.
  4. Κάντε κλικ στο κουμπί "-" για να μειώσετε προϊόντα στη παραγγελία.
  5. Κάντε κλικ στο "Μ" για να προσθέσετε σχόλια.
Παραγγελία
  1. Εμφανίζει παραγγελία, τα προϊόντα, τις πληροφορίες πληρωμής για το συγκεκριμένο τραπέζι.
  2. Κάντε κλικ στο προϊόν Για να επανεκτυπώσετε, ακύρωσετε, μεταφέρετε και να αλλάξετε τις τιμές.
  3. Κάντε κλικ στο κουμπί "μεταφορα" για να μεταφέρετε για να ανοίξετε / ελεύθερο τραπέζι.
  4. Κάντε κλικ στο "διαχωρισμός" κουμπί διάσπαση σειρά πολλαπλών λογαριασμών από το ποσό ή προϊόν.
  5. Κάντε κλικ στο κουμπί "πρόσωπο" για να ενημερώσετε τον αριθμό των ατόμων.
Πληρωμή
  1. Σαφώς δείχνουν subtoal, έκπτωση, φιλοδώρημα, φορολογικές, συνολικά, μετρητά και αλλαγή.
  2. Κάντε κλικ στο κουμπί "Εκτύπωση" για την παραλαβή εκτύπωσης.
  3. Κάντε κλικ στο "συρτάρι" για να ανοίξει συρτάρι μετρητών.
  4. Κάντε κλικ στο κουμπί "Πελάτης" για να προσθέσετε πελάτη στην παραγγελία.
  5. Κάντε κλικ στο κουμπί "Αποστολή" στο email παραλαβή.
  6. Κάντε κλικ στο κουμπί "Σημείωση" για να προσθέσετε σημείωση για την παραλαβή.
  7. Κάντε κλικ στο κουμπί "Ακριβώς" για τον καταβληθεί ως ίδια συνολικά.
