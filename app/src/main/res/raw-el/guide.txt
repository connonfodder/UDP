     Σύνδεση με το προεπιλεγμένο λογαριασμό: manager / 123456 
     Πριν από την έναρξη εκτέλεσης παραγγελιών, θα πρέπει να εγκαταστήσετε τα τραπέζια, τις κατηγορίες, τα προϊόντα και σχόλια για το εστιατόριό σας στις ρύθμισεις. 
     Για εκτύπωση αποδείξεων, θα πρέπει να ρυθμίσετε τον εκτυπωτή στο μενού ρυθμίσεων. Σιγουρευτείτε τη σωστή ρύθμιση του εκτυπωτή. 
     Μετά τη ρύθμιση, μπορείτε να ανοίξετε το τραπέζι, πέρνεται παραγγελίες, και να στέλνετε παραγγελίες στον κουζίνα / μπαρ εκτύπωση. 
     Αφού στείλετε παραγγελίες, μπορείτε να ελέγχετε τις παραγγελίες κάνωντας κλίκ στο τραπέζι, εξοφλήτε παραγγελίες και εκτυπώνετε αποδείξεις.

     Δωρεάν πίνακα 
         Κάντε κλικ στο τραπέζι για νέα παραγγελεία 
     Κάντε κλικ στο κουμπί του μενού της συσκευής, για reports, ρύθμισεις, βάση δεδομένων.

Παραγγελία 

     Κάντε κλικ στο όνομα του Προϊόντος παραγγελία για να αλλάξει τιμή, προσθέστε παρατήρηση και την εκμετάλλευση. 
     Κάντε κλικ στο "Επιβεβαίωση" για να επιβεβαιώσει τα προϊόντα.
     Κάντε κλικ στο κουμπί "+" για να προσθέσετε προϊόντα. 
     Κάντε κλικ στο κουμπί "-" για να μειώσετε προϊόντα στη παραγγελία. 
     Κάντε κλικ στο "Μ" για να προσθέσετε σχόλια.

Παραγγελία 
     Εμφανίζει παραγγελία, τα προϊόντα, τις πληροφορίες πληρωμής για το συγκεκριμένο τραπέζι. 
     Κάντε κλικ στο προϊόν Για να επανεκτυπώσετε, ακύρωσετε, μεταφέρετε και να αλλάξετε τις τιμές. 
     Κάντε κλικ στο κουμπί "μεταφορα" για να μεταφέρετε για να ανοίξετε / ελεύθερο τραπέζι. 
     Κάντε κλικ στο "διαχωρισμός" κουμπί διάσπαση σειρά πολλαπλών λογαριασμών από το ποσό ή προϊόν. 
     Κάντε κλικ στο κουμπί "πρόσωπο" για να ενημερώσετε τον αριθμό των ατόμων.

Πληρωμή 
     Σαφώς δείχνουν subtoal, έκπτωση, φιλοδώρημα, φορολογικές, συνολικά, μετρητά και αλλαγή. 
     Κάντε κλικ στο κουμπί "Εκτύπωση" για την παραλαβή εκτύπωσης.
     Κάντε κλικ στο "συρτάρι" για να ανοίξει συρτάρι μετρητών. 
     Κάντε κλικ στο κουμπί "Πελάτης" για να προσθέσετε πελάτη στην παραγγελία. 
     Κάντε κλικ στο κουμπί "Αποστολή" στο email παραλαβή. 
     Κάντε κλικ στο κουμπί "Σημείωση" για να προσθέσετε σημείωση για την παραλαβή. 
     Κάντε κλικ στο κουμπί "Ακριβώς" για τον καταβληθεί ως ίδια συνολικά.
