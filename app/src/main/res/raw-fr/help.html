Comment utiliser le programme?
  1. Ouvrir une session avec le compte par défaut: manager/123456
  2. Avant de prendre une commande, il faut encoder les tables, les catégories, les articles et les modifications dans les paramètres pour votre restaurant.
  3. Pour l'impression des reçus, vous devez configurer l'imprimante dans les paramètres. Assurez-vous de choisir la bonne imprimante.
  4. Après la configuration, vous pouvez ouvrir une table, prendre des commandes, les envoyer et les imprimer pour la cuisine et le bar.
  5. Après avoir envoyé les commandes, vous pouvez les contrôler par un appui long sur la table, clôturer la commande et imprimer le reçu.
Tables
  1. La couleur est différente pour une table occupée
    1. Appuyer sur une table sur l'écran.
  2. Table libre
    1. Appuyer sur une table pour l'ouvrir.
  3. Appuyer sur le bouton de menu pour le rapport, la configuration, la base de données.
Commande
  1. Montre la commande, les articles, les informations de paiement pour une table déterminée.
  2. Appuyer sur "confirmer" pour confirmer les articles de la commande.
  3. Appuyer sur "+" pour ajouter un article à la commande.
  4. Appuyer sur "-" pour diminuer un article de la commande.
  5. Appuyer sur "cercle rouge " pour ajouter une modification.
Commandé
  1. Montre la commande, les articless, les informations de paiement pour une table déterminée.
  2. Appuyer sur un élément de la commande pour ré-impression, annulation, transfert et modification de prix.
  3. Appuyer sur le bouton "Transfert" pour envoyer la commande vers une table occupée ou libre.
  4. Appuyer sur le bouton "Séparer" pour scinder la commande sur plusieurs factures par quantité ou élément.
  5. Appuyer sur le bouton "Client" pour mettre à jour le nombre de personnes.
Paiement
  1. Présente clairement le sous-total, la remise, les pourboires, la taxe, le total payé et la monnaie à rendre.
  2. Appuyer sur "Impression" pour imprimer le reçu.
  3. Appuyer sur "Tiroir" pour ouvrir le tiroir-caisse.
  4. Appuyer sur "Client" pour ajouter un client sur une commande.
  5. Appuyer sur "Email" pour envoyer le reçu par Email.
  6. Appuyer sur note pour ajouter une note sur le reçu.
  7. Appuyer sur "Exactement " pour un paiement au montant exact.
