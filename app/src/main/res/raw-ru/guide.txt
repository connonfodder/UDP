Как начать использование?
	Войдите с учетными данными: manager/123456
	Перед тем как оформлять заказы вам необходимо настроить столы, категории, позиции и модификаторы, это делается в настройках.
	Для печати чеков необходимо настроить принтер.
	После проведения всех настроект вы можете начать принимать заказы и отправлять их на печать на кухню или в бар.
	После отправки заказа вы можете открыть заказ для проверки, а затем напечатать чек.

Столы
	Занятые столы имеют другой цвет
		Нажмите на стол на экране
	Пустые столы
		Нажмите для открытия стола
	Нажмите на кнопку меню на вашем устройстве для перехода к настройкам
	

Предзаказ

	Нажмите на название позиции для ее редактирования
	Нажмите "Подтвердить" для подтверждения позиции
	Нажмите "+" для увеличения количества
	Нажмите "-" для уменьшения количества
	Нажмите на "Красный круг" для добавления модификатора

Заказ
	Показывает порядок, позиции, платежную информацию для конкретного стола
	Нажмите на позицию для печати, отмены, переноса или изменения цены.
	Нажмите "Перенос" для переноса заказа на другой стол
	Нажмите "Разделить" для раздела счета по позициям или по сумме
	Нажмите "Количество гостей" для изменения.

Оплата
	Показывает предитог, скидку, чаевые, налог, итог, оплату и сдачу.
	Нажмите "Печать" для печати чека
	Нажмите "Касовый бокс" для открытия касового бокса
	Нажмите "Плательщик" для добавления плательшика в заказ
	Нажмите "Email" для отправки чека на Email
	Нажмите "Заметка" для добавления примечания
	Нажмите "Без сдачи" при оплате без сдачи
