Каа начать использование
  1. Войдите с учетными данными: manager/123456
  2. Перед тем как оформлять заказы вам необходимо настроить столы, категории, позиции и модификаторы, это делается в настройках
  3. Для печати чеков необходимо настроить принтер
  4. После проведения всех настроект вы можете начать принимать заказы и отправлять их на печать на кухню или в бар
  5. После отправки заказа вы можете открыть заказ для проверки, а затем напечатать чек
Столы
  1. Занятые столы имеют другой цвет
    1. Нажмите на стол на экране
  2. Пустые столы
    1. Нажмите для открытия стола
  3. Нажмите на кнопку меню на вашем устройстве для перехода к настройкам
Предзаказ
  1. Нажмите на название позиции для ее редактирования
  2. Нажмите "Подтвердить" для подтверждения позиции
  3. Нажмите "+" для увеличения количества
  4. Нажмите "-" для уменьшения количества
  5. Нажмите на "Красный круг" для добавления модификатора
Заказ
  1. Показывает порядок, позиции, платежную информацию для конкретного стола
  2. Нажмите на позицию для печати, отмены, переноса или изменения цены
  3. Нажмите "Перенос" для переноса заказа на другой стол
  4. Нажмите "Разделить" для раздела счета по позициям или по сумме
  5. Нажмите "Количество гостей" для изменения
Оплата
  1. Показывает предитог, скидку, чаевые, налог, итог, оплату и сдачу
  2. Нажмите "Печать" для печати чека
  3. Нажмите "Касовый бокс" для открытия касового бокса
  4. Нажмите "Плательщик" для добавления плательшика в заказ
  5. Нажмите "Email" для отправки чека на Email
  6. Нажмите "Заметка" для добавления примечания
  7. Нажмите "Без сдачи" при оплате без сдачи
