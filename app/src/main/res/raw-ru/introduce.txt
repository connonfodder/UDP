<b>Restaurant POS - Point of Sale</b> полнофункциональное легкое в использовании POS приложение разработанное для ресторанов, баров, кафе и т.д. Приложение совместимо с различными кассовыми принтерами и кассовыми боксами, что позволяет экономить вам время и средства на установку других систем.<br><br>

<b>Приложение являетя автономным, без использования централизованного сервера, все данные хранятся на локальном устройстве.</b>

<b>Вы можете поддержать нашу работу по улучшению приложение купив расширенную версию программы.</b>

<b>Заказы</b>
• Открыть/закрыть стол
• Отменить/перместить/разделить заказ
• На вынос/обслуживание на месте
• Ожидание/приготовление позиции в заказе
• Переместить позицию в заказе

<b>Оплата</b>
• Разделить счет
• Чаевые/сервисный сбор

<b>Меню</b>
• Создать/изменить/удалить стол, категорию, позицию или модификатор
• Импорт/экспорт столов, категорий, позиций или модификаторов
• Несколько налогов на позицию
• Учитывать/исключить налог в цене позиции
• Печать заказов на кухонном/барном принтере
• График цен
• Поделиться меню через Dropbox™ или Google Drive™

<b>Принтер</b>(настройка принтера http://www.androidappshk.com/app/printer_setup_guide.pdf)
1. W&O POS принтер адаптер (installed Window PC)
• Поддержка всех типов принтеров с установленым PC драйвером
  (download WnO POS Adapter https://drive.google.com/#folders/0BxPQIRLGhJmvc1dXXzhQbnA1STQ)
2. Wifi/Lan и USB
• Поддержка множества терминальных принтеров с ESC/POS Command
   Проверенные модели принтеров:  EPSON TM-T88IV, TM-U220, STAR TSP743, CITIZEN CT-S651, Xprinter, GPrinter, OCOM, Rongta 

<b>Кассовый бокс(RJ12) </b>

<b>Счет на оплату</b>
• Логотип компании, название, адрес, номер телефона
• Имя клиента, адрес и телефон
• Заказаные позиции
• Общий счет, сдача
• Налоги
• Примечание
• Учитывать/исключить налоги
• Скидка
• Чаевые/сервисный сбор
• Руководство
• Нижний колонтитул

<b>Плательщик</b>
• Показывает информацию о плательщике в чеке на вынос
• Создает плательщика по номеру телефона
• Отправка счета на Email плательщика

<b>Отчет</b>
• Отчет по заказам
• Отчет по отмененным заказам
• Отчет по перенесенным заказам
• Отчет по позициям
• Отчет по модификаторам
• Отчет по сумме по категории
• Отчет по количеству по категории
• Отчет по оплате

<b>Разрешения</b>
• Менеджер/официант логин
• Отключение просмотра отчетов для официантов
• Отключение скидки для официантов
• Отключение сервисного сбора для официатнов
• Отключение отмены заказов
• Отключение автоматического открытия кассового бокса

<b>Настройки</b>
• поддержка национальной валюты
• Ежедненый отчет в конце дня
• Настраиваемое количество чисел после запятой
• Поддержка нескольких налогов
