Como usar?
  1. Entre com a conta padrão: manager/123456
  2. Antes de começar a receber pedidos você precisa configurar mesas, categorias, items e modificadores para o seu restaurante, em Configurações.
  3. Para imprimir recibos você precisa configurar uma impressora no menu de Configurações. Certifique-se de que a impressora está configurada corretamente.
  4. Após configurar, você pode abrir mesas, anotar pedidos e enviar impressões de pedidos para a cozinha ou bar.
  5. Após fazer pedidos você pode conferí-los clicando na mesa, pode fechar pedidos e imprimir comprovantes.
Mesas
  1. Mesa vaga
    1. Clique na mesa para abrir a tela de cardápio.
  2. Mesa ocupada colorida
    1. Clique na mesa para abrir a tela da mesa.
  3. Clique no botão de menu de dispositivo para ver relatórios, configurações e banco de dados.
Anotando pedidos
  1. Clique no nome do item para mudar preço, adicionar observação e espera.
  2. Clique "Confirmar" para confirmar o pedido dos itens.
  3. Clique "+" para incrementar a quantidade do item no pedido.
  4. Clique "-" para reduzir a quantidade do item no pedido.
  5. Clique "Caixa Verde" para adicionar modificadores.
Pedidos feitos
  1. Mostra pedidos, itens, informação de pagamento para a mesa específica.
  2. Clique no item do pedido para reimprimir, cancelar, transferir ou mudar o preço.
  3. Clique no botão "Transferir" para transferir o pedido para outra mesa.
  4. Clique no botão "Dividir" para dividir o pedido em múltiplos pagamentos por valor ou itens.
  5. Clique no botão "Convidados" para modificar o número de clientes à mesa.
Pagamento
  1. Mostra distintamente subtotais, descontos, gorjeta, impostos, total, valor pago e troco.
  2. Clique "Imprimir" para imprimir um comprovante.
  3. Clique "Gaveta" para a abrir a gaveta da caixa registradora.
  4. Clique "Cliente" para adicionar informações do cliente no pedido.
  5. Clique "Email" para enviar o comprovante por email.
  6. Clique "Observações" para adicionar anotações ao comprovante.
  7. Clique "Exato" para pagamento exato do valor total.
