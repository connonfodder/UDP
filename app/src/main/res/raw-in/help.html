Cara penggunaan?
  1. Masuk dengan akun awal: manager/123456
  2. Sebelum memulai pesanan,Kamu harus mengatur meja, kategori, Menu dan tambahan untuk pengaturan restaurant kamu.
  3. Untuk pencetakan nota, kamu harus mengatur printer kamu di pengaturan. Pastikan pengaturannya tepat.
  4. Sesudah pengaturan, Kamu dapat membuka meja, menerima pesanan, dan menerima pesanan dan mencetak di printer Bar/dapur.
  5. Setelah mengirim pesanan, kamu dapat memeriksa pesanan dengan meng klik lama di meja, Tutup pesanan dan mencetak nota.
Meja
  1. Meja makan berwarna
    1. Masuk ke meja di pengaturan.
  2. Meja bebas
    1. Masuk ke meja melalui meja.
  3. Masuk ke Menu, Untuk laporan, pengaturan, database.
Pemesanan
  1. Klik nama menu pesanan untuk mengubah harga, menambahkan catatan dan menahan.
  2. Klik "Yakin" untuk memesan menu.
  3. Klik "+" untuk menambah pesanan.
  4. Klik "-" untuk mengurangi pesanan.
  5. Klik "Lingkaran Merah" untuk menambahkan tambahan.
Dipesan
  1. Menampilkan Pesanan,Menu, Info pembayaran meja tertentu.
  2. Klik menu pesanan untuk mencetak ulang, batal, pindah dan mengganti harga.
  3. Klik "Pindah" Untuk memindah pesanan ke meja baru.
  4. Klik "Pisah" untuk memisah nota ke beberapa berdasarkan jumlah atau menu.
  5. Klik "Orang" untuk mengganti jumlah pengunjung.
Pembayaran
  1. Memperlihatkan total, diskon, tip, pajak, total, pembayaran dan kembalian.
  2. Klik "Cetak" untuk mencetak nota.
  3. Klik "Laci" untuk membuka laci kasir.
  4. Klik "Pelanggan" untuk menambahkan pelanggan ke pesanan.
  5. Klik "Email" untuk mengirim lewat email nota.
  6. Klik "Catatan" untuk menambahkan catatan pesanan.
  7. Klik "Pas" Untuk jumlah pembayaran sama dengan tagihan.
